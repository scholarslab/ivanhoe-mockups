The Ivanhoe Game

A WordPress Plug-in

The Ivanhoe Game is a plugin which allows for scholars and students to make critical interventions in a text, historical topic, work of art, or any given subject and collaboratively generate new understanding of the subject. Players select roles and make moves in response to other players or on their own. To see a sample game, select Demo. To browse all games on this installation, select Browse. To create a new game, select Create.
